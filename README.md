## Project Overview

This project focuses on developing a desktop application for managing an **air travel agency**. The application handles various tasks such as managing customers, flights, bookings, billing, and other related services. Built with **Java** and **SQL** for database management, the app aims to provide a seamless experience for both agents and travelers.
